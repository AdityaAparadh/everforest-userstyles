## About  

This project brings the **Everforest** color palette to websites via **userstyles**.  
It‚Äôs a fork of [Catppuccin userstyles](https://github.com/catppuccin/userstyles), but with the Everforest-style palette.  


---

## Goals  
1. **Everforest Everywhere** ‚Äì Apply the Everforest theme to popular websites.  
2. **Based on Catppuccin** ‚Äì Reuse their structure, scripts and styles for easier maintenance.
3. **Automated Conversions** ‚Äì Automation so future Catppuccin updates can be converted into Everforest.  

---

## Installation  

You‚Äôll need a userstyle manager like:  
- [Stylus (Firefox/Chrome/Edge)](https://add0n.com/stylus.html)  

### Steps:  
1. Install Stylus.  
2. Download [import.json](https://github.com/AdityaAparadh/everforest-userstyles/releases/download/rolling/import.json) from releases.
3. Open the Stylus ‚Äúmanage‚Äù page.
4. On the sidebar panel, click the Import button in the Backup section, and select the downloaded file from step 2.
